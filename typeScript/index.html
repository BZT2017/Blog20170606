# 简介

  • TypeScript 是 JavaScript 类型的超集，它可以编译成纯 JavaScript。
  • TypeScript 可以在任何浏览器、任何计算机和任何操作系统上运行，并且是开源的。

# 安装

  • 通过 npm（Node.js 包管理器） npm install -g typescript
  • 安装 Visual Studio 的 TypeScript 插件
